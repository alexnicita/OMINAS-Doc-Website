OMINAS - Opensource Multiple Instrument Analysis Software

OMINAS is an IDL-based software environment that allows you to reduce and analyze any data from space or planet based systems.

For information on acquiring a permanent or temporary IDL license, please visit IDL.

Funding for OMINAS' initial development was provided through the Cassini Mission.

OMINAS is designed to be used with data from any instrument.

With one simple translator for each instrument, OMINAS' entire analysis library can be applied to your data.

For more information on OMINAS' generality, please refer to the developer documentation.

OMINAS' core seperates geometry calculations and data anlysis.

This allows kernels from new missions to be smoothly added to OMINAS' library, which means that OMINAS was built to efficiently analyze data from multiple instruments.

OMINAS revolves around object oriented programming principles, optmizing the software for modularity and clean organization.

The developer documentation includes visuals of OMINAS' architecture.

OMINAS has a graphical tool (GRIM) that can be used alongside command line scripts for real-time data manipulation and analysis.

GRIM allows scientists to prototype systems and rapidly iterate changes while developing products, all while using the command line to script changes as needed.

Among all of this, OMINAS maintains speed comparable to C and portability on as many UNIX-based platforms as possible.
